# 📚 Library Management System

A simple web-based Library Management System built using Flask (Python).  
The project allows users to log in and manage books by adding, borrowing, and returning them.

---

# 🚀 Features

- 🔐 User Login System
- 📖 View Books
- ➕ Add Books
- 📘 Borrow Books
- 🔄 Return Books
- 🌐 Flask Web Application
- ☁️ Deployment Ready (Railway / Render)

---

# 🛠️ Technologies Used

## Backend
- Python
- Flask

## Frontend
- HTML
- CSS

## Deployment
- Gunicorn
- Railway / Render
